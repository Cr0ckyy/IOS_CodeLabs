Create a project - https://help.apple.com/xcode/mac/current/#/dev07db0e578

Build and run your app - https://help.apple.com/xcode/mac/current/#/devdc0193470

  1. Launch Xcode
  2. Click Create a new Xcode project

  1. Select iOS, App and click Next

  1. Enter the project information and click Next. Set the Interface as Storyboard, the Life Cycle as UIKit App Delegate and Language as Swift. Uncheck the boxes for Use Core Data and Include Tests. For the Organization Identifier, set it as sg.edu.rp.19XXXXXX where 19XXXXXX is your student number.

  1. Select a location to save the project and click Create

  1. Choose a scheme from the scheme pop-up menu in the toolbar

  1. Choose a run destination from the scheme pop-up menu.

  1. Click the Run button in the toolbar to build, run, and debug your app.

  1. The selected simulator will launch as a separate application.

About the main window - https://help.apple.com/xcode/mac/current/#/dev84c38774c

About the editor area - https://help.apple.com/xcode/mac/current/#/dev79c94bf05

  • Toolbar - for managing your project
  • Editor area - for viewing and editing the contents of your project including code, user interface files, property lists, project settings, and more.
  • Navigator area - for viewing the parts of your project including files, symbols, breakpoints, and build information.
  • Debug area - for controlling the execution of your app during debugging, and for displaying variables, register, and status information.
  • Inspector area - for viewing and editing information about the selected object in the navigator or editor area, or the entire project.
